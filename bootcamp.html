Guess Hue? - my first "aha" moment



IF statements, arrays, loops and objects. By week 3 into our program, we dove into JavaScript and our challenge was to build a word guess game. Something I learned early on was to give each of my projects a higher purpose. The purpose here? Learning the breadth of named colors HTML and CSS - all 140 up for guessing!

I could have used easy hints like "rhymes with" to help the user when guessing. Instead, I decided to use the color itself as a hint. What if we changed the background to reflect the color up for guessing?

That's how Guess Hue? was born. More importantly, what this app taught me was that from inception, I was practicing basic product management processes, such as feature development and MVP determination. See below for screen captures.
